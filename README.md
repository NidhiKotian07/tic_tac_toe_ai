#Intelligent Tic Tac Toe AI

This project implements an intelligent Tic Tac Toe game using two AI approaches(ML AI and Minimax AI).

#Features include-
1.Machine learning model to predict best move
2.Minimax Algorithm for optimal gameplay
3.Interactive GUI built with Tkinter
4.AI mode selection(ML vs Minimax)

#Technologies Used-
1.Python
2.Scikit-learn
3.NumPy 
4.Tkinter

#Run the game-
python modern_gui.py
